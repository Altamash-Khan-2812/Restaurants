# Eatwell

Eatwell allows food lovers to browse restaurants that others have recommended or share their own favorite dining spots. Whether you're seeking new places to try or want to highlight a hidden gem, Eatwell helps you make the world a tastier placeâ€”one recommendation at a time.

## Demo

Click <a href="https://restaurants-ib8y.onrender.com/">HERE</a> to see a live demo

## Features

- **Browse Restaurants**: Explore a curated list of community-shared restaurants.
- **Share a Restaurant**: Submit your own top picks and contribute to the community.
- **About Eatwell**: Learn about the purpose and vision behind the project.

## Tech Stack

- **Backend**: Node.js, Express.js
- **View Engine**: Ejs
- **Styling**: CSS
- **Deployment**: Render
